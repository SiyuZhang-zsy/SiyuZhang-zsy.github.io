# Siyu Zhang Academic Homepage

This repository contains a Jekyll-based academic personal homepage for Siyu Zhang.

The site is based on the [AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) template, with content adapted from Siyu Zhang's CV and layout simplified in the style of a classic academic homepage.

## Main Files

- `_config.yml`: site metadata and sidebar profile information.
- `_pages/about.md`: homepage content.
- `_data/navigation.yml`: top navigation links.
- `files/Siyu_Zhang_CV.pdf`: downloadable CV.
- `images/siyu-avatar.png`: simple profile placeholder image.

## Local Preview

Install the GitHub Pages/Jekyll dependencies, then run:

```bash
bundle exec jekyll serve
```

The site should then be available at `http://127.0.0.1:4000`.
